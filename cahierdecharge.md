# 📘 Cahier des Charges – Pharmacie en Ligne

## 1. 🧾 Présentation du Projet

Ce projet consiste à développer une application web de **pharmacie en ligne** permettant aux utilisateurs de consulter des produits pharmaceutiques, ajouter des articles au panier et passer des commandes en toute sécurité.

Le site sera développé avec les technologies suivantes :

* HTML / CSS
* JavaScript
* PHP
* MySQL

---

## 2. 🎯 Objectifs

* Mettre en place une plateforme simple pour consulter des médicaments
* Permettre aux utilisateurs d’acheter en ligne
* Gérer les commandes et les utilisateurs
* Offrir une interface moderne et responsive

---

## 3. 👥 Acteurs

### 👤 Client

* Créer un compte et se connecter
* Consulter les produits
* Ajouter au panier
* Passer une commande
* Consulter ses commandes

### 🛠️ Administrateur

* Gérer les produits (ajout, modification, suppression)
* Gérer les commandes
* Gérer les utilisateurs

---

## 4. ⚙️ Fonctionnalités

### 🔎 Catalogue Produits

* Affichage des produits
* Filtrage par catégorie
* Page détail produit

### 🛒 Panier

* Ajouter / supprimer des produits
* Modifier la quantité
* Calcul du total

### 📦 Commande

* Validation du panier
* Enregistrement dans la base de données
* Suivi du statut (en cours, validée, livrée)

### 🔐 Authentification

* Inscription
* Connexion / Déconnexion
* Gestion des sessions

### 🖥️ Espace Admin

* Gestion des produits (CRUD)
* Gestion des commandes
* Gestion des utilisateurs

---

## 6. 🔒 Contraintes Techniques

* Design responsive (mobile / desktop)
* Sécurisation des mots de passe (hash)
* Validation des formulaires (front + back)
* Connexion base de données avec PDO

---

## 7. 🖥️ Interfaces

* Page d’accueil
* Page produits
* Page détail produit
* Panier
* Page commande
* Authentification (login/register)
* Dashboard admin

---

## 8. 🚀 Améliorations (Bonus)

* Recherche avancée
* Système de notifications
* Paiement en ligne (simulation)
* Avis clients
* Chat support

---

## 9. 📌 Conclusion

Ce projet permet de créer une plateforme digitale facilitant l’accès aux produits pharmaceutiques, tout en offrant une expérience utilisateur simple, rapide et sécurisée.
